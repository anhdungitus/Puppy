Puppy.Cleaner

Project Created by Top Nguyen

This tool help Cleanup C sharp project

  • Remove Folders:

    • ".vs"
    • "packages"
    • "bin"
    • "obj"
    • "bld"
    • "Backup"
    • "_UpgradeReport_Files"
    • "ipch"
    • "Debug"
    • "Release"

  • Remove Files by extensions:

    • "*.suo",
    • "*.user"

Require .Net Framework 4.5.2
